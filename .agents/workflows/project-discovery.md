### 🔍 Passo a Passo: Project Discovery

1. **Análise de Estrutura**: Executar `list_dir` recursivamente para entender a hierarquia de pastas.
2. **Identificação de Stack**: Ler `package.json`, `tsconfig.json` e arquivos de configuração (`.env`, `next.config.js`).
3. **Mapeamento de Dependências**: Identificar bibliotecas principais (Auth, Banco de Dados, UI).
4. **Resumo de Escopo**: Comparar a estrutura atual com a `especificacao.md` para identificar o que falta.
5. **Relatório**: Criar um documento inicial de análise no diretório `Ref/`.
