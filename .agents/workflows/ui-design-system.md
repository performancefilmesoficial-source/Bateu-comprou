### 💅 Passo a Passo: UI Design System

1. **Paleta de Cores**: Definir variáveis CSS para as cores baseadas na especificação (Laranja, Azul Metálico).
2. **Tipografia**: Configurar a fonte Montserrat e escalas de tamanho (`h1`, `p`, etc).
3. **Tokens de UI**: Definir arredondamentos (`border-radius`), sombras (`box-shadow`) e espaçamentos.
4. **State Aesthetics**: Definir estilos para estados de Hover, Focus e Active.
5. **Guia de Estilo**: Criar/Atualizar `design_system.md` no diretório `Ref/`.
