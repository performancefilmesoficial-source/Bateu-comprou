### 🎨 Passo a Passo: Layout-First Planning

1. **Definição de Rotas**: Listar todas as páginas necessárias (`/`, `/dashboard`, `/auth/login`, etc).
2. **Wireframing**: Criar esboços descritivos ou em ASCII do layout de cada página.
3. **Componentização**: Identificar componentes repetitivos (Navbar, Sidebar, ProductCard).
4. **User Flow**: Mapear a navegação do usuário entre as telas.
5. **Aprovação**: Apresentar os planos de layout ao usuário antes de iniciar o desenvolvimento visual.
